 ----------------INSTALLATION----------------
| Requires python3			     |
| Extract and open terminal in the folder    |
| Run command -> ./requirements	             |
 --------------------------------------------
 
 ---------------------------------------------------------------HOW TO RUN--------------------------------------------------------------------
| Run python3 downlaod.py <filename.torrent> 											 	      |
| Arguments:1) '-d' - To view all details while downloading (peer messages such as choke,unchoke..,tracker responses..,etc)[default=No]	      |	
|	    2) '-m <number_of_max_peers>' - (Maximum number of peers to connect to. e.g. python3 download.py test.torrent -m 10)[default=15]  |
|	    3) '-top4' - (To use top 4 algorithm)											      |	
|	    4) '-seed' - (Seeding while downloading)											      |
 ---------------------------------------------------------------------------------------------------------------------------------------------
